# Como rodar o projeto

Este projeto foi limpo para envio. Para rodar, siga os passos abaixo:

1. **Instalar dependências:**
   Abra o terminal na pasta do projeto e rode:
   ```bash
   npm install
   # ou
   pnpm install
   # ou
   bun install
   ```

2. **Configurar variáveis de ambiente:**
   Crie um arquivo `.env.local` na raiz do projeto e adicione as chaves necessárias (peça ao proprietário do projeto se precisar).

3. **Rodar o projeto:**
   ```bash
   npm run dev
   ```

O projeto estará rodando em `http://localhost:3000`.
